Neste projeto, criei um conversor de unidades simples. Utilizei HTML para a estrutura da página e JavaScript para realizar a conversão de Celsius para Fahrenheit. Foi uma experiência para entender como manipular fórmulas de conversão e interagir com elementos de entrada em HTML.


In this project, I created a simple unit converter. I used HTML for the page structure and JavaScript to convert Celsius to Fahrenheit. It was an experience to understand how to manipulate conversion formulas and interact with input elements in HTML.
